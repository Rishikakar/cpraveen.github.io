Computational Methods (Jan-May 2016)

Class timings: Monday and Thursday, 11:00 AM to 12:30 PM
Class room: Auditorium
Grading: Homework (20), notes (20), midterm (30), final (30)

Class notes

You are expected to take notes in class. By every Monday morning, you must submit your class notes for the previous week. This will be considered for your assessment.

Codes

The codes used in the course can be obtained from github. If you use git, you should clone

git clone https://github.com/cpraveen/numpde

and change the branch to cm2016. Alternately, you can download a zip file.

Reference books

  1. Thomee and Larsson, Partial Differential Equations with Numerical Methods, Springer
  2. R. J. LeVeque, Finite Difference methods for ordinary and partial differential equations, SIAM
  3. P. Wesseling, Principles of Computational Fluid Dynamics, Springer
  4. E. Suli, Numerical solution of PDEs, lecture notes, search on his website or get it here
